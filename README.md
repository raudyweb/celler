# Celler Cal Puput

Web del restaurante Celler Cal Puput — Cocina catalana tradicional en Vilanova i la Geltrú.

## Stack

- React + Vite
- TypeScript
- Tailwind CSS + shadcn/ui
- Desplegada en Netlify

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy

La web se despliega automáticamente en Netlify al hacer push a `main`.
